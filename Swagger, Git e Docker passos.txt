Configuração do Docker (Windows)

0- Com o Docker instalado abra um Terminal PowerShell para o Banco de dados e cada Projeto abaixo.

1- Crie uma Rede (net)
Execute: 
	docker network create aviao-net

2- Baixar imagem H2/Database
Execute: 
	docker pull buildo/h2database

3- Criar Container
Execute: 
	docker run --name my-h2 --network aviao-net -p 8082:8082 -d buildo/h2database

4- Abra o Banco pelo Browser
No Browser: 
	http://localhost:8082/

5- Mude o campo JDBC URL
Insira: 
	jdbc:h2:my-h2
	
7- Remova username e password do Banco caso exista e clique em Test Connection
